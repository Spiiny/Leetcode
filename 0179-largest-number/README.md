<h2><a href="https://leetcode.com/problems/largest-number">179. Largest Number</a></h2>
<h3>Medium</h3>
<hr>

<p>Given a list of non-negative integers <code>nums</code>, arrange them such that they form the largest possible number and return it.</p>

<p>Since the result can be very large, return it as a string instead of an integer.</p>

<hr>

<p><strong>Example 1:</strong></p>

<pre>
Input: nums = [10,2]
Output: "210"
</pre>

<p><strong>Example 2:</strong></p>

<pre>
Input: nums = [3,30,34,5,9]
Output: "9534330"
</pre>

<hr>

<p><strong>Constraints:</strong></p>

<ul>
<li><code>1 &lt;= nums.length &lt;= 100</code></li>
<li><code>0 &lt;= nums[i] &lt;= 10<sup>9</sup></code></li>
</ul>

<hr>

<p><strong>Function Signature:</strong></p>

<pre>
string largestNumber(vector<int>& nums);
</pre>
